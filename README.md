## Table of Contents

- [Maintainers](#maintainers)
- [Contribute](#contribute)

## Usage

1. Clone this repo

2. Review outdated dependancies and upgrade if needed

```
yarn outdated
```

2. Install dependancies

```
yarn
```

3. Run app

```
yarn dev
```
